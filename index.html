My Skills

HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML elements tell the browser how to display the content.

CSS

CSS stands for Cascading Style Sheets. It is a style sheet language which is used to describe the look and formatting of a document written in markup language. It is generally used with HTML to change the style of web pages and user interfaces.

JAVASCRIPT

JavaScript is an open-source programming language. It is a text-based, lightweight, cross-platform, and interpreted scripting programming language. This language is very popular for developing web pages.

JQUERY

jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website. It is also used to add animation.

PYTHON

Python is a general-purpose language, object-oriented, high-level programming language with dynamic semantics. Python supports class inheritance. A class may have many subclasses but may only inherit directly from one superclass.

MySQL

MySQL stands for “Structured Query Language.” A relational database stores data in separate tables rather than putting all the data in one big storeroom. The database structure is organized into physical files optimized for speed.
